Installation & Configuration Guide

Step 1 : Go to Settings/ Users & Companies/ Users

Step 2 : Reload the page once done with the above configuration settings and you will see "Odoo Bill.com Bridge" in the menu

Step 3 : Configure your Instance by providing Location(URL), Username, Password, Organization Key and Developer Key

Step 4 : After filling up the details, to check if entered credentials are correct, click on the "Check Connection" button

Step 5 : On successful Test connection, a popup with success message gets displayed.
